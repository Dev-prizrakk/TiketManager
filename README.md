# TiketManager

Менеджер тикетов заказа — приложение для управления заявками/тикетами.

## Описание

TiketManager — это система для:

- создания тикетов пользователями,  
- управления статусами тикетов (открыт, в работе, закрыт и т.д.),  
- хранения истории тикетов,  
- назначения тикетов ответственным,  
- уведомления о новых тикетах и изменении статусов (в зависимости от конфигурации).

## Особенности

- Реализовано на **Java** с использованием Maven.  
- Чистая архитектура с разделением слоёв.  
- Поддерживает расширение функциональности: легко добавить новые статусы, роли, уведомления.  
- Лёгкий старт: достаточно настроить конфигурацию и запустить приложение.

## Требования

- Java 8+
- Maven
- Discord Token (для запуска)

## Установка и запуск

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/Dev-prizrakk/TiketManager.git
   cd TiketManager


2. Сборка проекта:

   ```bash
   mvn clean install
   ```

3. Настройте параметры (конфигурацию базы, свойства приложения) в `config.properties` или аналогичном конфигурационном файле. (Если такого файла нет — создайте и настройте под свой вариант.)

4. Запустите приложение:

   ```bash
   mvn exec:java
   ```

   Или запустите скомпилированный `.jar` (если настроена сборка в jar).

## Структура репозитория

* `src/main/java` — исходники Java-кода
* `src/main/resources` — ресурсы (конфигурация, файлы)
* `pom.xml` — Maven конфигурация

## Как пользоваться

* Администратор может создавать и назначать тикеты.
* Пользователи могут создавать тикеты через UI (если он реализован) или API.
* Для каждого тикета хранится история изменений статусов и ответственные лица.
* В дальнейшем можно интегрировать уведомления (например, email) при изменении статуса тикета.
